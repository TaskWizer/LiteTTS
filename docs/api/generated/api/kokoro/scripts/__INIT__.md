Kokoro TTS Scripts Package

This package contains utility scripts for the Kokoro ONNX TTS API,
including installation helpers, voice management tools, and
deployment utilities.

Available scripts:
- install_cpu_only: CPU-only dependency installation
- download_all_voices: Voice downloading utilities
- setup_complete_voice_system: Complete voice system setup
